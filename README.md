# 地方競馬AI ゆまちゃん

中央競馬版をベースに、地方netkeiba向けに作り替えた版です。

## 起動

```bash
pip install -r requirements.txt
python main.py
```

iPhoneから見る場合は、PC側で起動したあと ngrok などで `http://localhost:5000` を公開してください。

## 使い方

通常は以下を選んで実行します。

- 日付
- 地方競馬場
- レース番号
- 予算

地方netkeibaの race_id が分かる場合は、race_id を直接入力できます。

例:

- 園田 2026年4月29日 9R: `202650042909`
- 大井 2026年4月29日 11R: `202644042911`
- 高知 2026年4月25日 8R: `202654042508`

## 注意

地方netkeibaのHTML構造やアクセス制限により、出馬表・オッズ・過去走が取得できない場合があります。
オッズが取れない場合、買い目は「暫定評価」または「見送り」寄りになります。

## 主な変更点

- JRA競馬場コードを地方競馬場コードに変更
- `nar.netkeiba.com` / `nar.sp.netkeiba.com` 対応
- race_id直接入力に対応
- 地方騎手補正を追加
- 小回り・短距離・地方ダート向けの簡易コース補正を追加
- 未来情報漏れ防止のため、対象日以降の過去走は除外
